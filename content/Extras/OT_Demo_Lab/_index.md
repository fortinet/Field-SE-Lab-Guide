### **Introduction**

This guide is designed to provide Field SEs with a physical OT environment with Fortinet ruggedized equipment to demonstrate OT use cases. 

There are four main sections in this guide: 
1) **[Bill of Materials](Bill_of_Materials)** - List of materials required with URL's to purchase.
2) **[How-To Build the enclosure/case for this OT Lab](How_To_Build_Enclosure)** - describes the steps with pictures.
3) **[How-To Build the Arduino Modbus Devices for this OT Lab](How_To_Build_Modbus_Devices)** - describes the steps with pictures.
4) **[How-To Digital I/O for this OT Lab](How_To_Build_Digital_IO)** - describes the steps with pictures.
5) **[Demo Use Cases](How_To_Demo)**

This document should NOT be construed as the best and only way to build an OT Lab nor the only way to demonstrate using this OT Lab.
