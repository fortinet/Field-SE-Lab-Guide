Introduction

The intent of this Lab Build Guide is to significantly reduce the complexity and time required to setup the “infrastructure” around Fortinet technologies often needed in an SE Lab.

This Lab removes the need to:

  • plan and develop an IP Addressing scheme (this lab follows FNDN’s)
  • learn, understand and implement WAN Emulation (this lab utilizes TCGUI)
  • plan and install multiple “end user” end points (this lab uses Ubuntu)

Following this lab’s step-by-step instructions, the result is the deployment and base config for everything shown in the diagram below, at which point an SE can spend time learning and utilizing Fortinet technologies.
